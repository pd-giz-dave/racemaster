Getting Started

  1. Click Open Data Folder to select your race data directory.
  2. Set up the event details in Event Settings.
  3. Optionally, import pre-entries via Pre-Entries (SI).
  4. Register competitors on the Entries page.
  5. Record finish times on the Finishers page.
  6. Generate Results & Prizes when racing is done.
  7. Print forms and results from Print Forms.
