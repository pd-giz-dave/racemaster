Getting Started

  1. Navigate to Datasets to log in (optional) and select or create a dataset.
  2. Set up the event details in Event Settings.
  3. Optionally, import pre-entries via Pre-Entries (SI).
  4. Prepare pre-race paperwork from Paperwork.
  5. Register competitors on the Entries page.
  6. Record finish times (and start times) on the Finishers page.
  7. Optionally, import SI results via SI Results
  8. Generate Results & Prize List when racing is done.
  9. Check for outstanding entrants on the Safety Check page.

Important — Print Settings

This app completely controls the print layout. When the print dialog opens you must set Margins to None. Using any other margin setting will shift the content on the page.

 

This is necessary to ensure the printed forms are properly aligned irrespective of the browser/printer used.
